# Parchís de la Doble R 🎲🚀

¡Bienvenido al **Parchís de la Doble R**! Una aplicación educativa diseñada específicamente para ayudar a niños de 5 años (y más) a dominar la lectura de palabras y frases con el fonema "rr" de una forma divertida y lúdica.

## 🎯 Objetivo del Juego
El objetivo es ser el primer jugador en llevar su ficha desde la **Casa** hasta la **Meta** (la copa 🏆). Por el camino, el niño deberá leer frases y trabalenguas que refuerzan la pronunciación y comprensión de la "doble r".

## 🛠️ Cómo Funciona

### 1. Inicio de la Partida
- **Selección de Jugadores:** Puedes elegir entre **2, 3 o 4 jugadores**. Cada uno tiene un color: Rojo, Verde, Azul o Amarillo.
- **Modos de Juego:**
  - **Tablero Rápido (24 casillas):** Ideal para sesiones cortas de lectura (línea recta).
  - **Tablero Completo (68 casillas):** El desafío tradicional del Parchís para una práctica más extensa.

### 2. El Turno
- El jugador activo toca el **Dado**. El dado muestra puntos y el número para facilitar el reconocimiento numérico.
- **Salir de Casa:** Cualquier número del 1 al 6 permite sacar la ficha de casa a la primera casilla.
- **Movimiento:** La ficha saltará automáticamente casilla por casilla según el número obtenido.

### 3. El Desafío de Lectura 📖
- Al terminar el movimiento, aparecerá un **Modal de Lectura**.
- El niño debe leer la frase o trabalenguas en voz alta.
- Todas las frases tienen entre **6 y 8 palabras** para mantener un nivel de dificultad adecuado.
- Una vez leída, se pulsa el botón **"¡LO LEÍ!"** para pasar el turno al siguiente jugador.

### 4. Ganar la Partida
- El primer jugador que llegue exactamente a la última casilla (Meta) será el ganador y verá el trofeo de victoria.

## 📚 Contenido Educativo
- **Más de 90 textos únicos:** Incluye frases sencillas ("El perro corre por el barro") y trabalenguas clásicos.
- **Sin repeticiones:** El juego utiliza una lógica de "bolsa" para asegurar que no se repitan las frases hasta que se hayan leído casi todas.
- **Enfoque Fonético:** Todo el contenido está centrado en palabras con **"rr"** (perro, carro, torre, cerro, etc.).

---
*Diseñado para fomentar la autonomía en la lectura sin ayudas de voz, obligando al niño a decodificar el texto por sí mismo.*
