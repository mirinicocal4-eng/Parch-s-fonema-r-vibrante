# Parchís de la Doble R 🎲🚀

Este proyecto fue creado y puede ser editado en **[Google AI Studio Build](https://ai.studio/build)**.

## 💻 Desarrollo Técnico

### Cómo trabajar localmente

Si deseas ejecutar este proyecto en tu propia máquina, sigue estos pasos:

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```
   La aplicación estará disponible en `http://localhost:3000`.

3. **Construir para producción:**
   ```bash
   npm run build
   ```

### Tecnologías utilizadas
- **React 19** con **TypeScript**
- **Vite** como herramienta de construcción
- **Tailwind CSS** para los estilos
- **Motion** para las animaciones de las fichas y el dado
- **Lucide React** para la iconografía
